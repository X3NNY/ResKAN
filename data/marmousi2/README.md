# Marmousi2 Dataset

The Marmousi2 dataset files are too large to be included in this repository. Please download them manually and place the following files in this directory:

- `MODEL_DENSITY_1.25m.segy` - Density model (SEGY format)
- `MODEL_P-WAVE_VELOCITY_1.25m.segy` - P-wave velocity model (SEGY format)
- `seismic_0_30.npy` - Synthetic seismic data (NumPy array format, can be generated using `scripts/forward_model.py`)
